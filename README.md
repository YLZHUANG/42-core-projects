# üìò Project Summaries

### **Libft** ‚Äî C Standard Library Reimplementation  
Foundation library reimplementing essential libc functions (`mem`, `str`, linked lists).

### **ft_printf** ‚Äî Custom Printf  
Reimplementation of `printf` with formatting support.

### **get_next_line** ‚Äî File Reader  
Line-by-line file reading with buffered I/O.

### **push_swap** ‚Äî Sorting Algorithm Challenge  
Optimized sorting using two stacks and constrained operations.

### **minitalk** ‚Äî UNIX Signals IPC  
Client/server communication using `SIGUSR1` and `SIGUSR2`.

### **minishell** ‚Äî Custom Shell  
A minimal shell featuring pipelines, parsing, environment management, built-in commands, and process control.

### **fract-ol** ‚Äî Fractal Renderer  
Interactive fractal renderer (Mandelbrot, Julia, Burning Ship). Using MiniLibX library for window rendering.

### **cub3d** ‚Äî Raycasting Engine & Virtual Museum  
Wolfenstein-like raycasting engine built in C.  

### **Philosophers** ‚Äî Concurrency Problem  
Dining Philosophers with threads, mutexes, and deadlock prevention.

### **NetPractice** ‚Äî Network Configuration  
Exercises in IP addressing, subnetting, routing.

### **Born2beroot** ‚Äî Linux System Administration  
Debian-based VM with user management, SSH, UFW, password policies, monitoring.

### **Inception** ‚Äî Docker Infrastructure  
Multi-container WordPress setup with Nginx and MariaDB using Docker Compose.

### **webserv** ‚Äî HTTP/1.1 Web Server  
Nginx-like web server in C++ handling CGI, routing, chunked transfer, and sockets.

### **CPP Modules (00‚Äì09)** ‚Äî OOP Foundations  
Classes, inheritance, polymorphism, templates, STL, exceptions, RTTI.

### **ft_transcendence** ‚Äî Full-Stack Web App (Final Project)  
Real-time multiplayer Pong + chat + authentication  
**Tech stack:** NodeJS, TypeScript, SQLite, Fastify, JWT, WebSockets, OAuth2, 2FA authentification. 

### **snow-crash** ‚Äî Cybersecurity challenge
Intro to enumeration, privilege escalation & light binary exploitation.

### **override** ‚Äî Cybersecurity challenge
Stack overflows, format string vulnerabilities, and low-level exploitation.  

### **rainfall** ‚Äî Cybersecurity challenge
Advanced exploitation including ROP, ASLR bypass, and memory corruption.
